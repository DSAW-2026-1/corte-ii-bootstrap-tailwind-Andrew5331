## Overview
This project is a responsive Stats Preview Card Component built using HTML and CSS. It was developed in two versions: one using Bootstrap and another using Tailwind CSS.

The component includes a heading with highlighted text, a descriptive paragraph, three statistics, and an image with a violet overlay effect. The layout adapts to different screen sizes, stacking vertically on mobile devices and displaying in two columns on desktop screens.

The project follows a predefined style guide, including specific colors, typography, and spacing, demonstrating responsive design and modern front-end development practices.
